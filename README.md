📌 Project Overview

The High Cloud Airline Data Analysis Dashboard is a comprehensive Power
BI solution designed to analyze airline operations, passenger traffic,
and performance metrics.

It provides insights into flight volumes, passenger trends, load
factors, route performance, and airline efficiency.

------------------------------------------------------------------------

🎯 Objective

-   Analyze airline performance across routes and carriers
-   Track passenger and flight trends over time
-   Evaluate load factor and seat utilization
-   Understand distance-based flight distribution
-   Identify top-performing airlines and routes

------------------------------------------------------------------------

🛠️ Tech Stack

-   Power BI Desktop
-   Power Query
-   DAX
-   Data Modeling

------------------------------------------------------------------------

📂 Dataset Description

The dataset includes airline operational data such as:

-   Airlines & Carriers
-   Flights & Routes
-   Passenger volumes
-   Available seats & load factor
-   Distance groups
-   Flight types
-   Time dimensions
